# Kernel
A kernel written with the intention to learn about OS developement.

## Dependencies
- i686-gcc8
- Make
- WSL (if you are on windows)
- Xming server (if you are on windows)
- Qemu (to boot the kernel)

# Building
Building + running the kernel.
```sh
make
```
s
Cleaning (build dir.)
```sh 
make clean
```
## Running
```sh
make run
```
